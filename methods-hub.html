Root-Finding Methods

Bisection Method

Find roots using interval bisection. Always converges but is slower.

False Position

Regula falsi method for faster root finding. Often faster than bisection.

Secant Method

Approximation method using secant lines. Faster convergence.

Gauss-Seidel Method

Iterative method for solving linear systems with fast convergence.

Gauss-Jacobi Method

Iterative approach for linear systems using parallel computations.
